# PowderSense Screenshots

This directory contains screenshots of the PowderSense web interface for documentation purposes.

## Hardware Components

Located in `docs/images/`:

**bourns_potentiometer.png**
- Bourns PTA6043-2015DPB103 slide potentiometer
- 10kΩ linear, 60mm travel
- Shows full component with mounting holes
- White plastic housing with black slider
- Reference for BOM and assembly guide

**steel_rod_threaded.png**
- 4mm diameter steel rod (85mm length)
- M4 threading visible on one end
- Precision ground silver steel
- Shows finished rod ready for assembly
- Reference for rod preparation in assembly guide

**product_render.png**
- 3D CAD render of assembled device
- Shows complete unit with display
- Green PETG enclosure
- Used in README header

**product_photo.png**
- Real photo of working PowderSense unit
- Mounted on reloading press
- Display showing measurement (4.999 grains)
- "PERFECT" status indicator
- Used in README header and marketing

## Web Interface Screenshots

### Dashboard

**dashboard_main.webp**
- Main dashboard view
- Current measurement display (large)
- Last measurement panel
- Quick settings panel with profile selector
- Session statistics
- Real-time chart area
- Control buttons (Measure, Calibrate, etc.)

**dashboard_advanced.webp**
- Lower section of dashboard
- Session logs table
- Advanced settings panel
- System information
- Recent measurements table
- Configuration management buttons
- Firmware update controls

### Configuration

**profile_dialog.webp**
- Add New Configuration dialog
- Input fields for:
  - Configuration Name
  - Caliber
  - Bullet Weight
  - Powder Name
  - Target Grain
- Save and Cancel buttons

### WiFi Setup

**wifi_config.webp**
- WiFi configuration page (AP mode)
- SSID input field
- Password input field
- Current status display
- Save & Connect button
- Instructions for after saving
